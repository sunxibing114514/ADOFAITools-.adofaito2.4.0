# 使用方法
1. 上传.adofai文件并点击转换。下载转换完毕文件
2. 替换部分文字（可使用[文本替换](https://sunxibing114514.github.io/text-replace/)）
       + true替换为"Enabled"
       + flase替换为"Disabled"
       + 下载
3. 将seetings项中
       + "Enabled"替换为true
       + "Disabled"替换为false
4. 将null替换为0
5. 详见seetings替换.txt
